      CSS

      CSS引入方式

      1.内联样式：在标签内部的style属性里面写样式代码，弊端：不能复用

      2.内部样式：在head标签里面添加style标签在标签体内写样式代码，好处是能在当前页面内复用样式， 弊端：不能多页面复用

      3.外部样式：在单独的css文件中写样式代码，在页面的head标签里面通过link标签引入css文件，好处：可以多页面复用

      在style标签中只能用css，此时注释为/* */

      选择器：选择页面中指定元素，并将声明快中的样式应用到选择器对应元素上、

      声明块：紧跟在选择器后，使用｛｝包括，名值对结构，分号结束

      CSS引入方式的优先级

      - 内联优先级最高

      - 内部和外部优先级相同，后执行的会覆盖前面执行的。

      CSS的选择器

      \1. 标签名选择器 标签名{} 需要选择页面中所有某种标签时使用

      \2. id选择器 #id{} 需要选择页面中某一个元素时使用

      \3. 类选择器 .class{} 需要选择页面中某一类元素时使用

      \4. 分组选择器 #abc,.xyz,h1{} 页面中多个选择器对应的元素需要统一设置样式时

      \5. 属性选择器 标签名[属性名=’属性值’]{}

      \6. 子孙后代选择器 div p{} 获取div里面出现的所有的p标签

      \7. 子元素选择器 div>p{} 获取div里面的下一级(子元素)p标签

      \8. 伪类选择器 用于选择元素的状态

      - 未访问状态： a:link

      - 访问过状态: a:visited

      - 鼠标悬停状态: a:hover

      - 点击状态: a:active

      \9. 任意元素选择器 *{}

      布局相关样式（盒子模型）

      1.控制元素在页面显示的位置和大小

      2.盒子模型包括： 宽高、外边距、内边距、边框

      1) 宽高： width、height

      ​ 块级元素可以修改宽高、行内元素不可以直接修改宽高只能有内容决定宽高

      2) 外边距：指元素距上级元素边框的距离或相邻兄弟元素边框的距离

      ​ 块级元素外边距全部生效，行内元素上下不生效

      • 当元素的上边缘和上级元素的上边缘重叠时，给元素添加上外边距会出现粘连问题，在上级元素中添加 overflow:hidden
      • 块级元素上下相邻，同时添加上外边距和下外边距，取最大值
      • 行内元素左右相邻，同时添加左外边距和右外边距，两者相加

      \3. 边框： border，可以给元素的任意方向添加边框

      • 块级元素边框全部生效，会影响元素所占宽高
      • 行内元素边框全部生效，会影响元素的宽但不影响元素的高

      内边距

      1.元素边框距内容的距离

      2.如果移动元素的文本内容，只能通过修改内边距的方式，记住修改内边距会影响元素的宽高

      3.如果移动元素的子元素，可以有两种方式：

      1)给大的添加内边距（会影响元素的宽高） 2)给小的添加外边距（推荐）

      4.块级元素内边距全部生效，会影响元素的宽高

      5.行内元素内边距全部生效，会影响元素所占宽度但不影响高度

      文本相关样式

      1.设置对齐方式 text-align:left/right/center

      2.文本修饰 text-decoration:overline/underline/line-through/none

      3.文本颜色 color:red

      4.文本阴影 text-shadow:颜色 x偏移值 y偏移值 模糊度值越小越清晰

      5.设置圆角 border-radius: 10px 值越大越圆

      字体相关样式

      1.字体大小 font-size:10px;

      2.字体加粗 font-weight:bold/normal;**

      3.斜体 font-style:italic

      4.字体名称 font-family:字体名 可以写多个，能用哪个用哪个

      溢出设置 overflow

      1.visible 显示(默认)

      2.hidden 隐藏 可以解决粘连问题

      3.scroll 滚动显示

      元素的显示方式 display

      1.块级元素默认值为 block, 独占一行 可以修改宽高

      2.行内元素默认值为 inline, 共占一行 不能修改宽高

      3.行内块 inline-block, 共占一行 可以修改宽高。

      行内元素垂直对其方式vertical-align

      1.上对齐 top

      2.下对齐 bottom

      3.中间对齐 middle

      4.基线对其 baseline(默认)
